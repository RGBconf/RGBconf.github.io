RGB 1st Games Programming Conference: Red for Indie Edition

The RGB 1st Games Programming Conference: Red for Indie Edition is a game tech conference for independent game developers.

Our goal for the first edition is to gather game developers from Switzerland and Auvergne-Rhône-Alpes region for a day of talks and networking.

Location: Octagon

Dates: October 28th 2023

Schedule:

  • Welcome: 8:30am-9:30am CEST
  • Opening keynote: Elias Farhan: 9:30am-10:00am CEST
  • Tom Avatars: Crafts and synesthesia: making a video-game album and its concert version: 10:00am-10:45am CEST
  • Florian Golesti: Unreal Gameplay Ability System - Showcase: 10:45am-11:30am CEST
  • David Stark: Procedural Map Generation in Airships: Conquer the Skies: 11:30am-12:15am CEST
  • Lunch break: 12:15pm-1:30pm CEST
  • Nicolas Regnault: Game architecture: empowering designers to create a small dense simulated open world: 1:30pm-2:15pm CEST
  • Julien Revel: A minimalistic but flexible AI and scenario system for Unity: 2:15pm-3:00pm CEST
  • Coffee break: 3:00pm-3:30pm CEST
  • Oleg Loshkin: Overview of Audio for Video Games: 3:30pm-4:15pm CEST
  • Sébastien de Graffenried: Designing a UI system from scratch: 4:15pm-5:00pm CEST
  • Closing keynote: Elias Farhan: 5:00pm-5:30pm CEST
  • Evening show by Tom Avatars: starting from 6:00pm CEST
